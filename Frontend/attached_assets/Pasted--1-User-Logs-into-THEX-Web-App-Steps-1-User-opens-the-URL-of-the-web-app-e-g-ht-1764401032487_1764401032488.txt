## **8Ô∏è‚É£ Test Execution (Dockerized Selenium Grid)**

**Steps:**

1. For each **browser selected**:
    
    - Starts **Selenium node session**.
        
2. For each **website/page**:
    
    - Executes all uploaded scripts.
        
    - Performs checks:
        
        - Wait for element
            
        - Assert text
            
        - Click buttons
            
        - Execute custom script logic

    - Screenshots (on failure or all steps if configured)

    - Logs (pass/fail, console outputs)
        

**Backend Notes:**

- Parallel execution: Browser √ó Site √ó Script run concurrently.
    
- Reduces total execution time.
